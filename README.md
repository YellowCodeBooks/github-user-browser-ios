# GitHub User Browser – iOS (TymeX Mobile Programming Test)

## Built with
- Swift + SwiftUI
- MVVM architecture
- Combine
- URLSession

## Features
- Paginated list of GitHub users (20 per fetch)
- Detail screen for each user
- Offline support with Core Data (planned)
- Unit tests with XCTest
